Generate category and product category descriptions automatically using OpenAI's API in multiple languages with customizable length and tone.

== Description ==

Webby for WordPress: Category Description is a powerful plugin that allows you to generate high-quality, SEO-friendly category descriptions with a single click using OpenAI's advanced language models.

= Features =

* Generate category descriptions with AI in seconds
* Support for WooCommerce product categories
* Support for 10 different languages
* Customizable description length (short, medium, long)
* Customizable tone options (standard e-commerce, professional, friendly, personal)
* Bulk generation for all product categories at once
* Dedicated settings page for API configuration
* Works with both new and existing categories
* Respects parent-child category relationships
* Simple and intuitive user interface
* Secure API key storage

= How It Works =

1. Configure your OpenAI API key in the plugin settings
2. Go to any category or product category edit page
3. Select your preferred language, length, and tone
4. Click the "Generate Description with AI" button
5. The AI-generated description will be automatically inserted into the description field

= Bulk Generation =

For WooCommerce stores with many product categories, you can now:

1. Go to the Product Categories page
2. Use the bulk generation tool at the top of the page
3. Select your preferred language, length, and tone
4. Click "Generate All Descriptions"
5. Watch as descriptions are generated for all your product categories at once

== Installation ==

1. Upload the `webby-category-description` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Webby Category to configure your OpenAI API key
4. Start generating category descriptions!

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, you need an OpenAI API key to use this plugin. You can get one from [OpenAI's website](https://platform.openai.com/api-keys).

= How much does it cost to use? =

The plugin itself is free, but you'll need to pay for OpenAI API usage. OpenAI charges based on the number of tokens processed. For category descriptions, the cost is typically very low (fractions of a cent per description).

= Can I edit the generated descriptions? =

Yes, the generated descriptions are inserted into the standard WordPress category description field, so you can edit them just like any other content.

= What are the different length options? =

* Short: 1-2 sentences, perfect for concise category descriptions
* Medium: 2-3 sentences, balanced for most category pages
* Long: 4-5 sentences, ideal for detailed category descriptions

= What are the different tone options? =

* Standard E-commerce: Neutral, professional tone suitable for most online stores
* Professional: Formal and business-oriented tone
* Friendly: Warm and approachable tone
* Personal: Conversational and engaging tone

= Can I generate descriptions for all my product categories at once? =

Yes! For WooCommerce stores, you can use the bulk generation tool on the Product Categories page to generate descriptions for all categories in one go.

= Does it work with custom taxonomies? =

The plugin works with the default WordPress category taxonomy and WooCommerce product categories. Support for additional custom taxonomies may be added in future versions.

= Is my OpenAI API key secure? =

Yes, your API key is stored securely in the WordPress database using WordPress's built-in options API.
